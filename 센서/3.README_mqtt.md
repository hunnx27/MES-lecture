# 사전 준비
### 모스키토 설치
- mosquitto 2.0.22 버전 설치
- https://mosquitto.org/download/
- mosquitto-2.0.22-install-windows-x64.exe 다운로드
- 설치하면 자동으로 mosquitto 라는 이름으로 서비스 등록되어 있음

### 모스키토 설치 확인
- 작업관리자 > 서비스 > 목록에서 mosquitto 서비스 확인 가능, 상태가 "실행 중" 인지 확인할 것
- cmd에서 다음명령으로 기본 포트인 1883이 열려있는지 확인
```bash
netstat -ano | findstr 1883
>>>>
  TCP    127.0.0.1:1883         0.0.0.0:0              LISTENING       
  TCP    [::1]:1883             [::]:0                 LISTENING       
```

### 모스키토 실습
```bash
# 구독하기
[mosquitto 설치경로]\mosquitto_sub -h [주소] -p [포트] -t [주제]

# 메시지 발행(보내기)
[mosquitto 설치경로]\mosquitto_pub -h [주소] -p [포트] -t [주제] -m [메시지]

# ex)
"C:\Program Files\mosquitto\mosquitto_sub" -h localhost -p 1883 -t mes_topic
"C:\Program Files\mosquitto\mosquitto_pub" -h localhost -p 1883 -t mes_topic -m "hello MES!!!"
```
